# Project Phoenix v1.0

A mobile-first, installable Progressive Web App for the Project Phoenix training program.

## Deploy with GitHub + Vercel

1. Create a GitHub repository named `project-phoenix`.
2. Upload all files from this folder to the repository.
3. In Vercel, choose **Add New → Project**.
4. Import `cyrusaskin/project-phoenix`.
5. Select framework preset **Other**.
6. Leave build command and output directory blank.
7. Deploy.
8. Open the Vercel URL in Safari on iPhone.
9. Tap **Share → Add to Home Screen**.

## Included

- Premium mobile UI
- Nine workouts across three weeks
- Home, workout, and progress views
- Warm-up checklists
- One-exercise-at-a-time workout flow
- Set logging
- Rest timer
- Shoulder pain tracking
- Local session storage
- Offline caching

## Data

All data is stored locally in the browser. Clearing Safari website data or changing devices will remove saved logs.
